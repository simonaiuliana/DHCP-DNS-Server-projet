# DHCP / DNS - Windows Server 2019

## 📌 Description
Mise en œuvre et configuration des services réseau DHCP et DNS sur Windows Server 2019. Ce projet permet l’automatisation de l’attribution des adresses IP via DHCP ainsi que la gestion et la résolution des noms de domaine dans un réseau local.

---

## ⚙️ Technologies utilisées
- Windows Server 2019
- DHCP Server Role
- DNS Server Role
- VMware 

---

## 🛠️ Étapes réalisées
- Installation du rôle DHCP
- Configuration d’un scope DHCP
- Activation du serveur DHCP
- Installation et configuration du rôle DNS
- Création de zones DNS
- Test de résolution de noms
- Attribution automatique des IP

---

## 📸 Captures d’écran
- Configuration DHCP
- Zone DNS
- Test `ipconfig /renew`

---

## 🎯 Objectif du projet
Comprendre la gestion d’un réseau local et l’automatisation des services IP et DNS dans un environnement Windows Server.
